### ⚡ Reach
> **LinkedIn Recruiter Outreach Automation**

Scrapes recruiter emails from LinkedIn and fires off personalized emails with your resume attached — fully automated, one click.

**Highlights:**
- 🤖 **Puppeteer** browser automation — real Chrome, not HTTP
- 🎯 Anti-bot spoofing — user-agent + webdriver flag hidden
- 📧 **Nodemailer** Gmail SMTP with resume attachment
- 🔍 Stable selector strategy using HTML `autocomplete` attrs
- ⌨️ Tab-key trick to handle LinkedIn's React form focus
- 🛡️ Zero credential storage — all in-memory
